# 联调测试：会员详情模块

## 一、接口

- `GET /api/member/{id}/detail`

返回结构包含三部分：
- 会员基础信息（余额/积分/累计消费/等级/状态）
- 最近账务流水 `recent_account_logs`
- 最近订单 `recent_orders`

## 二、页面联调步骤

1. 登录后进入“会员中心”
2. 点击任意会员行的“详情”
3. 页面弹出详情信息（当前使用 alert 展示）
4. 核对基础信息与列表一致
5. 核对最近账务流水与最近订单展示正常

## 三、验收点

- 余额、积分、累计消费显示正确
- 充值后可看到 `RECHARGE` 流水
- 余额支付订单取消后可看到 `ORDER_CANCEL` 流水
- 最近订单状态与订单管理页一致
